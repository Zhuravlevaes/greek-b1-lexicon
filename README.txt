Greek B1 Lexicon — PWA

Как запустить локально на компьютере:
1. Распакуйте архив.
2. В папке выполните:
   python3 -m http.server 8000
3. Откройте http://localhost:8000

Как установить на iPhone:
1. Опубликуйте содержимое папки через HTTPS (GitHub Pages, Netlify, Cloudflare Pages и т.п.).
2. Откройте ссылку в Safari.
3. Нажмите «Поделиться» → «На экран Домой».

Важно:
- PWA не устанавливается из file:// напрямую.
- Данные и прогресс хранятся локально в браузере.
- Периодически используйте «Резервная копия».
- Новые слова добавляются через JSON-обновления.
